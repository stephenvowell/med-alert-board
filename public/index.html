Connect your ring

Your Oura app credentials are configured locally in .env. Make sure your Oura developer app uses this redirect URI:

http://localhost:3000/auth/callback

  1. In cloud.ouraring.com, confirm the redirect URI above matches exactly
  2. Run npm install and npm start
  3. Click Connect Oura below and approve access
  4. If some metrics are missing, click Disconnect then connect again to grant all permissions
